# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-01-17

### Changed
- Migrated entire project from JavaScript to TypeScript
- Converted all `.js` and `.jsx` files to `.ts` and `.tsx`
- Added TypeScript configuration (`tsconfig.json`)
- Added type definitions for all components and utilities

### Added
- Set up Jest testing framework with TypeScript support
- Added initial test suite for components
- Configured Jest with `jest.config.ts`

## [0.1.0] - Previous Release

### Added
- Initial release of Alone Against the Zone Companion
- Character creation and management
- Dungeon grid drawing and navigation
- Dice rolling functionality
- Encounter tracking
- Dark mode support
